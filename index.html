Projects

Databases Class Automation Tool

This is a tool residing on the Saunders College of Business faculty website which I developed for professors teaching database classes at RIT. The page allows users to create a Databases Class entry, which automatically generates a master database and phpmyadmin login on either our MySQL or MariaDB server. The professor can then add students manually one at a time or in bulk by importing an Excel. The master account has access to all databases in the class while student accounts can only view their own.

Vanden Brul Display Screen

The purpose of this project was to create a display interface for the Saunders College of Business at RIT that displayed all past winners of the Vanden Brul award. I worked as the sole developer on this project and built the display using HTML, JavaScript, and CSS within the CodeIgniter 4 framework.

Phil Saunders Timeline

This project revolved around making a timeline for Phil Saunders that could be shown on a touch screen display in the RIT college of business building. The CSS, HTML, and JavaScript was written entirely by hand without the use of any frontend frameworks.
